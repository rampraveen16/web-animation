Creating a Keyframe Animation

The animate function takes two parameters, an array of KeyframeEffects and AnimationEffectTimingProperties options.

Syntax
 var element = document.getElementById("animate");
 var animation = element.animate(keyframeEffect,timeline);
KeyframeEffects

The target KeyframeEffect to assign to the animation.

var animation = element.animate([
{ transform: 'scale(1)', opacity: 1 },
{ transform: 'scale(.5)', opacity: .5 },
{ transform: 'scale(.667)', opacity: .667 },
{ transform: 'scale(.6)', opacity: .6 }
]);
timeline

Specifies the timeline with which to associate the animation.

var animation = element.animate([], {
    duration: 700, //milliseconds
    easing: 'ease-in-out', //'linear', a bezier curve, etc.
    delay: 10, //milliseconds
    iterations: Infinity, //or a number
    direction: 'alternate', //'normal', 'reverse', etc.
    fill: 'forwards' //'backwards', 'both', 'none', 'auto'
  });

Basic Animation

var ball = document.getElementById("ball");
 ball.animate([
 { bottom:'0px',height: '90px', offset: 0 },
 { bottom:'0px',height: '160px', offset: 0.1 },
 { bottom:'50%',height: '160px', offset: 1 }
 ],
 { duration:1000,
 iterations:Infinity,
 fill:'forwards',
 direction:'alternate',
 easing:'ease-out',
 delay:0
 });

Animation Methods

Animation.cancel()

Clears all keyframeEffects caused by this animation and aborts its playback.

Animation.finish()

Seeks either end of an animation, depending on whether the animation is playing or reversing.

Animation.pause()

Suspends playing of an animation.

Animation.play()

Starts or resumes playing of an animation, or begins the animation again if it previously finished.

Animation.reverse()

Reverses playback direction, stopping at the start of the animation. If the animation is finished or unplayed, it will play from end to beginning.
